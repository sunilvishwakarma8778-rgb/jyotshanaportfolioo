# Jyotshana Shahi — Portfolio Website

A beautiful animated portfolio website built with React + Vite.
Color theme: White · Pink · Blue

## 🚀 How to Run

1. **Install Node.js** from https://nodejs.org (version 18 or higher)

2. **Extract** this zip folder anywhere on your computer

3. **Open Terminal / Command Prompt** inside the folder

4. **Install dependencies:**
   ```
   npm install
   ```

5. **Start the development server:**
   ```
   npm run dev
   ```

6. **Open your browser** and go to: `http://localhost:5173`

## 🏗️ Build for Production (to host online)

```
npm run build
```
This creates a `dist/` folder — upload it to any hosting site (Netlify, Vercel, GitHub Pages).

## ✨ Features
- Smooth scroll animations
- Animated skill bars
- Floating blobs background
- Responsive (mobile + desktop)
- Pink/Blue/White theme
- Playfair Display + DM Sans fonts
